专业知识基本功

Posted on 2019-06-18 | Edited on 2019-06-27 | Views:
Python开发

  • 控制语句（if, for, while, pass, assert, with, yield, import, in）

    • assert 断言，如果assert后的表达式判定为否，则抛出程序异常
    • with 替代传统的try…finally
    • yield yield就是 return 返回一个值，并且记住这个返回的位置，下次迭代就从这个位置后(下一行)开始。

  • 类型（object, list, tuple, dict, set）

  • 表达式（lambda, 列表推导式）

    • lambda

    • 列表推导式

      ​ 根据已经存在的列表创建新的列表，但不限于列表，元组集合也可以被推导

  • 函数（map, reduce, filter, zip, sort, enumrate, isinstance）

    • map

      以指定函数对列表进行操作，返回的结果为map类。

    • reduce

      Python3中需要导入reduce函数，可以将列表中的各个元素进行迭代

    • filter

      filter(函数，列表)，可以对列表进行指定函数的过滤，返回值为列表

    • sort

      students为类对象的list，用sorted进行比较时可以自己定cmp函数，例如这里要通过比较第三个数据成员来排序，代码可以这样写：

    • enumrate()

      ​ 将列表指定为可以遍历的对象。

    • isinstance

      isinstance() 与 type() 区别：

      • type() 不会认为子类是一种父类类型，不考虑继承关系。
      • isinstance() 会认为子类是一种父类类型，考虑继承关系

  • 面向对象（类成员、实例化、构造函数、析构函数、继承、派生、多态、super）

    • 析构函数

      当使用del 删除对象时，会调用他本身的析构函数，另外当对象在某个作用域中调用完毕，在跳出其作用域的同时析构函数也会被调用一次，这样可以用来释放内存空间。

  • 数值计算（numpy、scipy、pandas、sklearn、gensim）

    • numpy

      • numpy.array

      • array 的创建

        • zeros、ones函数

        • full函数

        • arange函数

          与 python 中的range相同，但在range()中的第三个参数有所增强，第三个参数可以使用小时的形式进行，小数的递进。

        • linspace函数

  - randint函数

  - seed函数

    将numpy.random.seed(value) 设为固定的值，之后的random.randint所产生的随机数将固定

  - normal函数

- **array 合并和分割**

将两一维数组进行横向合并

  x = np.array([1, 2, 3])
  y = np.array([3, 2, 1])
  numpy.concatenate([x,y])

  #二位数据合并

默认为竖直叠加合并，更改concatenate的第二个参数axis=1，可使其横向叠加合并

  A = np.array([[1, 2, 3],
                [4, 5, 6]])
  np.concatenate([A, A])

  #当维度不同的矩阵合并时，使用vstack竖直叠加、hstack水平叠加
