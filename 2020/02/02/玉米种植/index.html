玉米的种植

玉米属于一年生高大草本植物。原产中美和南美洲是重要的粮食作物。玉米营养成分全面，富含优质碳水化合物、维生素和微量元素，同时蛋白质含量较高食用后饱腹感强，因此是世界重要的粮食作物。

)

在农业生产中有一个名词叫做“源库关系”。简单的理解玉米穗是库，玉米的根系和叶片是源。玉米叶片合成的有机物以及根系吸收的肥水形成了源，向玉米穗部供应形成果实。源能够充分的满足库，玉米的穗部籽粒就会饱满、充实；如果源不能满足库，就会出现凸尖、瘪粒、空粒以及籽粒饱满度变差等情况的发生。

从玉米的生理特性来讲，多穗型也就是多个玉米棒子的植株，这样的品种是不符合稳产、高产的生理需求。一般一株玉米可以结一至八个玉米棒子，但通常源是很难满足库的需求，人们在种植玉米的过程中会摘除一些玉米穗保留1-2个玉米棒子，以保证玉米棒长得更大、玉米粒更饱满。

玉米的产量

每个420平方米的标准篮球场，可以种植2000多株玉米，这个篮球场每年就可以收获大约3000个玉米棒子。根据联合国粮农组织的数据，2017年中国玉米产量2亿6千万吨（是香蕉产量的20倍），这包括用于制作玉米面粉、玉米油、罐头和我们市面上见到的玉米。

)

如此高产的作物是如何种植的呢？

品种选择

如何选好玉米良种，是关系到秋季产量增收的关键问题。常见的选择依据包括当地热量资源条件，日照时间充足，平均温度较高，就尽量选择生长期较长的玉米品种，使优良品种的玉米生长潜力得到充分发挥。

优先选择纯度高的玉米品种。玉米品种纯度的高低和质量的好坏直接影响玉米的产量。对于玉米种子纯度每下降1%，产量就会下降0.61%。选用高质量高纯度品种的种子是玉米高产的有利保证。

此外，在选种过程中还需要考虑当地的降水、土地肥沃程度、病虫害等因素。

种植技巧

  1. 玉米很难容忍霜冻和寒冷的气候条件，在低于16ºC的环境下是无法种植玉米的。如果外面气候温暖可以在室外进行播种，最安全的播种方法是在温室、暖房的室内进行播种。将种子播种在1.5-2.5英寸（5厘米）深的有机土壤中，过深或过浅播种玉米种子都不利于玉米幼苗的发芽和生长。

)

  1. 大约经过一周左右的时间，待幼株长到6英寸 15厘米左右高时，准备将它们种植到室外，小心的将幼苗一株一株分开，并将其播种到温暖的室外土壤中，在移植的过程中要保证玉米植株间距不低于45厘米，深度同样保持在2-2.25英寸之间，相同的深度可以让植株平等的竞争土壤资源。尽管玉米是十分耐旱的作物，但在极端干燥炎热的环境下，要一周浇水一次保证土壤湿润。

    )

  2. 夏末时期，玉米的流苏和穗轴开始形成。大约经过六周的时间，末尾的流苏变成黑褐色，玉米粒中饱含汁水时，玉米棒就可以采摘了。
